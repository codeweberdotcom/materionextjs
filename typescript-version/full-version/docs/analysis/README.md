# Анализы и исследования

Эта папка содержит все анализы проекта.

## Структура

- **`ui-ux/`** - Анализы пользовательского интерфейса и опыта использования
- **`performance/`** - Анализы производительности
- **`security/`** - Анализы безопасности
- **`architecture/`** - Архитектурные анализы

## Правила

1. Выбрать соответствующую категорию для анализа
2. Использовать формат имени: `analysis-[категория]-[модуль]-[описание]-[дата].md`
3. Если анализ выявил проблемы - создать соответствующие документы в `improvements/`
