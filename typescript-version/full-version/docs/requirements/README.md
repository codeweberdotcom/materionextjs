# Технические задания (ТЗ)

Эта папка содержит все технические задания проекта.

## Структура

- **`active/`** - Активные ТЗ, которые находятся в работе
- **`completed/`** - Завершенные ТЗ
- **`archived/`** - Архивные ТЗ (старые или отмененные)

## Правила

1. При создании нового ТЗ - поместить в `active/`
2. При завершении ТЗ - переместить в `completed/`
3. При отмене или устаревании - переместить в `archived/`
4. Использовать формат имени: `requirement-[модуль]-[описание]-[дата].md`
