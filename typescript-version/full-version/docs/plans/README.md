# Планы работ

Эта папка содержит все планы работ проекта.

## Структура

- **`active/`** - Активные планы, которые находятся в работе
- **`completed/`** - Завершенные планы
- **`roadmap/`** - Долгосрочная дорожная карта проекта

## Правила

1. При создании нового плана - поместить в `active/`
2. При завершении плана - переместить в `completed/`
3. Использовать формат имени: `plan-[модуль]-[описание]-[дата].md`
4. Связать план с соответствующим ТЗ в `requirements/`
