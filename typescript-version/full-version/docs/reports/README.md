# Отчеты о выполнении

Эта папка содержит все отчеты проекта.

## Структура

- **`weekly/`** - Еженедельные отчеты о работе
- **`monthly/`** - Ежемесячные отчеты
- **`testing/`** - Отчеты по тестированию
- **`deployment/`** - Отчеты о развертывании

## Правила

1. Выбрать соответствующую категорию для отчета
2. Использовать формат имени: `report-[тип]-[модуль]-[период].md`
3. Обновить статусы связанных планов/ТЗ после создания отчета
