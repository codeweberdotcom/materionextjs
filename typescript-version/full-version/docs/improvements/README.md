# Улучшения и проблемы

Эта папка содержит все документы о проблемах и улучшениях проекта.

## Структура

- **`bugs/`** - Известные баги
- **`technical-debt/`** - Технический долг
- **`enhancements/`** - Предложения по улучшению функционала

## Правила

1. Выбрать соответствующую категорию:
   - **`bugs/`** - для ошибок и багов
   - **`technical-debt/`** - для технического долга
   - **`enhancements/`** - для улучшений функционала
2. Использовать формат имени: `improvement-[тип]-[модуль]-[описание]-[дата].md` или `bug-[модуль]-[описание]-[дата].md`
3. При исправлении - создать документ в `fixes/` и обновить статус

## Шаблон улучшения/бага

См. [DOCUMENTATION_STANDARDS.md](../DOCUMENTATION_STANDARDS.md#5-улучшениебаг)

## Статусы

- **Открыт** - проблема обнаружена, требует решения
- **В работе** - проблема решается
- **Тестирование** - решение готово, проходит тестирование
- **Закрыт** - проблема решена и протестирована
